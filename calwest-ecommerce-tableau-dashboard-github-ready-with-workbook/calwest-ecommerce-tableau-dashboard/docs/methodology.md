# Methodology

## Business objective

Analyse Calwest e-commerce performance across revenue, profit, discounts, customers, products, promotions and US regions.

## Dashboard suite

### 1. E-Commerce Performance Overview

- KPI cards for revenue, profit, discount, profit margin, orders, customers, units, quantity and average order value
- Monthly revenue or profit trends by category
- Category and subcategory performance
- Product profitability scatter plot
- Order-date filtering
- Parameter-driven switching between revenue and profit

### 2. Regional Performance Summary

- US state map grouped by region
- Promotion versus non-promotion revenue share by month
- Category and subcategory treemap
- Top-10 products by revenue
- State selection actions to filter supporting visuals

## Validation

The Python script independently checks record counts, key metrics, category performance, regional performance, promotion share and product profitability using the same source tables.

# Publication checklist

Before publishing this repository:

- [ ] Add the author's completed `.twbx` workbook under `tableau/`.
- [ ] Add an original exported dashboard screenshot under `assets/`.
- [ ] Confirm that no course-provider completed workbook is included.
- [ ] Confirm that raw customer names and email addresses are not included.
- [ ] Check all workbook data connections use relative or parameterised paths.
- [ ] Confirm dashboard titles, tooltips and filters are readable.
- [ ] Add the Tableau Public URL to the README after publication.

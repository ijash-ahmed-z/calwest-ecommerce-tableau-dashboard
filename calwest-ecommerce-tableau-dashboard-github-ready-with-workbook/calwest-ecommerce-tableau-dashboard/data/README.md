# Source data

The Tableau case study uses the following local source files:

- `factOrder.csv`
- `dimCustomer.csv`
- `dimLocation.csv`
- `dimPayment.csv`
- `dimProduct.csv`
- `dimPromo.csv`
- `dimShipping.csv`

These files were supplied as part of course materials and are **not redistributed in this public repository** because no public reuse licence was included. Place authorised local copies under `data/raw/` to run the validation script.

The script uses customer data only to calculate aggregate counts. It does not export names or email addresses.
